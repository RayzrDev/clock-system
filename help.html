Setup
Setting up Clock System is surprisingly easy. Simply press the more_vert button in the top right corner, go to Settings, and then press Sign In. Once you have signed in, just select the spreadsheet you would like to send data to.
Usage
In order to track a user's clock in/out time, simply have them enter the name in the main input box and either press the Enter key on the keyboard or press Clock In. Once they have clocked in, it will start tracking the start time of when they clocked in. You can have as many users as you wish clocked in at once. In order to clock out, select your name from the list if it is not already selected, and press Clock out
